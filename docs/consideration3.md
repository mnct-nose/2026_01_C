# 体格と耐久の関係についての考察
 
## 重さは防御力そのものではない
 
軽いポケモンでも防御力が高いものは多く、単純に「重いほど防御力が高い」とは言えない。
ただし、400kgを超える超重量級になると、防御力が100〜150の範囲に収束していく傾向が見られる。
 
## デカさはタフさではない
 
高さ20m級の巨体でも、HPは150程度が上限に近い。
一方で、ハピナスは高さ1.5mでありながら全ポケモン最高のHP255を誇る。
体格よりも「生命力」というゲーム内の設定が、HPの外れ値として顕著に表れている。
 
## グラフを突き抜ける外れ値が存在する
 
10cmでありながら1トン近いコスモウムや、1.6mで0.1kgのゴーストなど、現実の物理法則を無視した図鑑設定が、散布図などのデータ上で極端な外れ値を生み出す原因となっている。
